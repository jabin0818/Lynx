<h3 align="center">基于React+SpringBoot的多功能工具箱系统</h3>

>基于 Vue+SpringBoot 进行开发，提供多种不同的知识分享和学习方式，项目分为博客、卡片、视频、动态、标签、用户和创作中心模块。

## ✨ 功能

- 博客笔记Markdown语法编辑
- 提供多种不同的知识分享和学习方式
- 发表博客、笔记、卡片集、视频和动态等进行知识分享
- 按照学习进度复习笔记卡片
- 社区类应用点赞、评论、动态、私信、关注等功能


## 🔧 技术栈

- Vue
- Element UI
- Canvas
- Axios
- Vuex
- Less
- WebSocket
- WangEditor
- Mavon-editor
- SpringBoot
- MySQL
- Redis
- ...


## 📦 安装

```bash
# 安装依赖
npm i

# 启动服务
npm run serve
```
